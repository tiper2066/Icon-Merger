## ICON Merger 프로젝트

### 개요

사용자가 메인 아이콘과 병합할 아이콘을 각각 선택하고 병합(Merge) 버튼을 누르면, 2개의 아이콘을 병합해서 새로운 아이콘으로 만들어 주는 서비스가 메인 기능이다.  


### 기능

1. 메인 및 병합용 아이콘 목록을 보여줌
2. 메인 및 병합용 아이콘을 각각 추가, 선택, 선택 해제, 전체 선택, 전체 해제, 삭제 할 수 있음
3. 메인 및 병합용 아이콘을 병합하여 최종 병합 아이콘을 생성함
4. 최종 병합 아이콘에 대해서 색상, 선 두께, 크기를 설정할 수 있음
5. 최종 병합 아이콘을 png, jpg, svg 중에 선택해서 다운로드할 수 있음
6. 메인 및 병합용 아이콘은 각각의 아이콘 업로드 다이얼로그를 통해서 아이콘을 업로드할 수 있음 (프로젝트 > docs > ui-image.jpg 이미지 하단의 다이얼로그 UI 참고)

- 메인 및 병합용 아이콘 반드시 SVG 포맷으로 업로드해야 하며, 드래그하거나 클릭하여 선택할 수 있음 
7. 변경된 아이콘 속성 설정은 초기화 기능을 제공한다  


### UI

- UI 디자인과 레이아웃 구현은 프로젝트 > docs > ui-image.jpg 를 참고한다.
- UI는 작업 순서를 고려해서 배치하도록 한다.   


1. 전체 화면을 2개의 컬럼으로 분리한다. 3:1 비율로 구성한다. (좌측 75%, 우측 25%)  

2. 좌측 컬럼은 다시 하위에 동일한 너비의 2 컬럼으로 분리하고 메인 및 병합용 아이콘 리스트를 보여준다.  

3. 우측 컬럼에는 병합 결과 미리보기와 간단한 설정, 그리고 병합 아이콘을 다운로드 할 수 있는 패널을 배치한다.  

4. 좌측 컬럼의 메인 아이콘 리스트에서 메인 아이콘을 선택하고 병합용 아이콘 리스트에서는 병합할 아이콘을 선택한다.  

5. 좌측 컬럼에서 메인 및 병합할 아이콘을 모두 선택하면 우측 컬럼의 패널에서 병합된 결과의 아이콘을 미리보기 영역에 보여준다.  

6. 우측 컬럼의 패널에서는 병합된 아이콘의 색상, 두께, 크기를 변경하고 원하는 포맷을 선택한 다음 다운로드 버튼을 클릭해서 최종 병합된 아이콘을 다운로드할 수 있다.  


- 색상은 10개의 선택지가 있으며 작은 박스로 2줄에 걸쳐 표시해 준다. 색상 박스 각각의 테두리는 배경색 보다 약간 진한색으로 설정한다.
- 선 두께는 1.5px 간격으로 최소 0.5px 에서 최대 3px까지 설정할 수 있게 한다.
- 크기는 최소 16px 부터 최대 256px 까지 4px 간격으로 설정할 수 있게 한다.
- 다운로드 포맷은 png, jpg, svg 중에서 선택 가능하게 한다.  


### 색상 설정 값

- 사용자에게 아래 10개의 색상을 선택지로 제공한다. 
- 만일 사용자가 흰색을 선택했다면 아이콘 리스트의 아이콘 배경은 검정색, 아이콘은 흰색으로 표현해야 한다.

1. RGBA (0, 96, 169, 1)
2. RGBA (48, 43, 207, 1)
3. RGBA (12, 115, 239, 1)
4. RGBA (45, 166, 250, 1)
5. RGBA (93, 214, 213, 1)
6. RGBA (221, 82, 76, 1)
7. RGBA (254, 204, 9, 1)
8. RGBA (153, 155, 158, 1)
9. RGBA (0, 0, 0, 1)
10. RGBA (255, 255, 255, 1)

- 아이콘이 흰색일 경우, 색상 박스의 테두리는 (229, 231, 235, 1) 로 한다.
